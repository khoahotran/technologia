1. Get Brand API
***Brand Response:

1. Get All Brands
API: http://localhost:8082/api/brands
Response:
List<BrandResponse>


2. Get Brand By id
API: http://localhost:8082/api/brands/{brand_id}
Response: 
{
    "status": 200,
    "data": {
        "brandId": 1,
        "name": "Anker"
    },
    "message": "Get brand by id successfully!"
}


3. Get paging brands
API: http://localhost:8082/api/brands/paged
Params:
* page: int (default 0)
* size: int (default 10)
* sortBy : name, id (default)
* sortDirection: ASC, DESC (default)
Response:
{
    "status": 200,
    "page_number": 0,
    "page_size": 2,
    "count_items": 4,
    "count_pages": 2,
    "data": List <BrandResponse>,
    "message": "Get brands successfully"
}

4. Search and Filter API


API: http://localhost:8082/api/products/search-filter
params ( tất cả đều optional) gồm:
* page: bắt đầu từ 0 (default 0)
* size: (default 10)
* sortBy: gồm name, id/product_id, price/display_price, rating/average_rating. create_at (default)
* sortDirection: ASC / DESC (default)
* minPrice : đơn vị VNĐ
* maxPrice: đơn vị VNĐ //search dựa trên display price
* keyword: tìm trong tên và description
* minRating
* maxRating
* categoryId
* brandId
